# C3-P01 — Example: The Key and the Door
**Status:** draft
**Group:** chapter-03

## Assumed knowledge
- **Walk**
- **Push** — from chapter 1.
- **Run** — from chapter 2.

## What this puzzle teaches
- **Point-and-click: examine/use** — clicking a loose item picks it up into inventory,
  and a held item can be used on a target object (key on door). This is the new idea.

## Mechanics used
- Point-and-click: examine/use
- Walk

## Setup
A single room. A key lies on the floor. A locked door blocks the exit. Nothing else.

## Intended solution
1. Walk to the key and click it to pick it up into inventory.
2. Walk to the door and use the key on it.
3. Exit through the opened door.

## Known alternate solutions
- None. The door is the only exit and only the key opens it.

## Failure states
- None. The key cannot be lost or made unreachable in this layout, so there is no
  soft-lock and nothing to fail. Intentionally clean for a first inventory lesson.
